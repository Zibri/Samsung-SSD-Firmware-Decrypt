# Samsung-SSD-Firmware-Decrypt
Samsung SSD Firmware Decrypt

# Install 7zip first!

# Compile it!
```$ make```
# Example
```
$ wget "https://semiconductor.samsung.com/resources/software-resources/Samsung_SSD_980_PRO_5B2QGXA7.iso"
$ ./sfwd.sh Samsung_SSD_980_PRO_5B2QGXA7.iso
```
